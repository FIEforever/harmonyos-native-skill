## 添加截图

1. 从 DevEco Studio 或其他工具截取上述图片
2. 将图片保存到 `screenshots/` 目录
3. 重命名为对应的文件名
4. README.md 中的图片引用会自动生效

## 图片规格

- 格式: PNG 或 WebP
- 建议宽度: 800-1200px
- 命名: 使用英文小写 + 连字符

## 注意

GitHub 仓库中不含实际截图文件，仅包含占位说明。如需完整截图，请自行添加。
